# Memexia - 自主生长的思想宇宙
### 1. 愿景与起源
Memexia 的名字与灵感来源于计算机先驱万尼瓦尔·布什（Vannevar Bush） 在1945年提出的 Memex 构想。布什预见了基于“关联索引”而非层级分类的知识未来，并描述了可存储、链接并形成个人“思想轨迹”的设备。Memexia 旨在用当代人工智能与网络技术，彻底实现并超越这一伟大预言。

### 2. 是什么
Memexia 是一个开源的思想实验引擎与知识网络自动化平台。它模拟并扩展了人类思维的联想过程，将静态的知识库转化为一个活的、能自主繁衍的有机体。

起点：用户提供一个初始概念（如“语言塑造思维”）。

过程：系统通过大语言模型进行逻辑推导，生成关联概念；同时通过向量嵌入计算语义相似度，与网络中的既有节点建立连接，甚至形成循环论证或发现隐藏模式。

产出：最终生成一个复杂的、非线性的思想图谱。这张图谱持续演化，每一次探索都可能触发新的生长分支。

### 3. 核心价值

对个人：作为终极的“第二大脑”，它能突破个人认知惯性，自动帮你发现知识盲区，建立跨领域洞见。

对研究者与创作者：提供自动化的文献综述辅助、研究问题生成与创意灵感激发。

对开源社区：作为一个可编程、可扩展的协议，它鼓励社区共同定义思想的“生长算法”，探索人机协作思考的新范式。

### 4. 技术栈与开源精神
项目将主要采用现代Python技术栈（如LLM API/本地模型、向量数据库、图计算引擎、Web可视化框架），并在 GNU AGPLv3 协议下开源。此协议确保了任何基于Memexia的改进都必须回馈社区，捍卫其作为公共知识基础设施的开放属性，防止其被闭源商业化。

### 5. 长期目标
Memexia 的终极目标并非构建一个封闭的应用，而是提供一套让思想得以自主组织、生长和连接的基础协议。我们相信，思想的未来不在于更庞大的数据库，而在于更丰富、更动态、更具启发性的连接。
